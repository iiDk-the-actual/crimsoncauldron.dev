// github authentication

Providing a Personal Access Token lets this page make authenticated API requests to GitHub. You can skip this, but you may be rate limited.

⚠ Unauthenticated: 60 requests/hour per IP.
Authenticated: 5,000 requests/hour.
Each page load makes multiple requests per fork — you can hit the limit quickly without a token.
How to get a token (no permissions needed):
  1. Go to github.com/settings/tokens
  2. Click Generate new token → choose Fine-grained or Classic
  3. For Classic: set any expiry, select no scopes (public data only)
  4. For Fine-grained: set expiry, leave all permissions at No access
  5. Click Generate token and copy it below
Token is stored in sessionStorage only — never sent anywhere except GitHub.

// frequently asked questions
What is an instance?
Due to the archival of ii's Stupid Menu, it's hard to find reliable forks of said mod. This website serves as a host for the forks to communicate.

These forks are hosted and continued by other people, and I have no affiliation with the users or any post additions to the database.
How do I add my instance?
  1. Make an account on GitHub.
  2. Make a fork of ii's Stupid Menu.
  3. Maintain the repository, amass stars, and stay updated to get to the top of the list.
Who is maintaining this site?
This site is not maintained. It shall update based on GitHub rankings and score. Results shown may be inaccurate, malicious, or outdated. Use your judgement when interacting with any of the listed repositories. Always check the code and be cautious when downloading or running software from the internet.
